게시글 상세 구현 과정
1. 게시글 리스트에서 한 게시글의 제목을 클릭하면 이동할 url을 설정
 => url : /board/detail
 => list.jsp에서 a태그 href에 적용

2. 컨트롤러에 /board/detail를 처리할 수 있도록 작업

3. /board/detail에 대한 화면부분인 jsp를 추가

4. list.jsp에서 a태그에 /board/detail로 게시글에대한 정보를 같이
   넘겨줘야함. 기본키인 번호를 넘겨줌

http://localhost:8080/spring/board/detail?num=?

5. 화면에서 넘겨준 게시글정보(num)를 컨트롤러에서 받기 위해
   매개변수 num를 추가

6. 컨트롤러가 서비스에게 번호(매개변수)를 넘겨주면서 해당 번호와 
   일치하는 게시글(리턴타입)을 달라고 요청
7. 컨트롤러는 서비스가 준 게시글을 화면에 전송

8. 서비스는 다오에게 번호와 일치하는 게시글을 DB에서 가져오라고
   일을 시키고, 서비스는 가져온 게시글을 컨트롤러에게 전달

9. 매퍼는 번호와 일치하는 게시물을 가져오는 쿼리문을 작성

10. 서버에서 보내준 게시글 정보를 화면에 출력
