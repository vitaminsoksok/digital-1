게시글 리스트 구현 과정

1. 컨트롤러는 서비스에게 게시글 전체를 가져오도록 요청하고,
   서비스는 컨트롤러에게 게시글 전체를 가져다 준다.
  =>서비스에게 알려줄 정보 : 없음 => 매개변수
  =>서비스가 컨트롤에게 알려줄 정보 : 게시글 전체 => 리턴타입

2. 서비스는 다오에게 게시글 전체를 가져오도록 요청하고,
    다오는 서비스에게 게시글 전체를 가져다 준다
 - 서비스는 다오가 준 게시글 전체를 컨트롤러에게 전달한다.

3. 매퍼는 게시글을 검색해서 전달하는 쿼리문을 작성한다
 -  select 이용

4. 컨트롤러는 서비스가 준 게시글 전체를 화면에 전달

5. 화면에서 게시글 전체를 출력
